# SSO
SSO (Single Sign On) back end system for user data.

## How to Use
- [x] Create the JSON files as endpoint in `app folder`.

## How to Build
- [x] Open terminal
### Build for all environments
- [x] Run command `make`
### Build for Linux environments
- [x] Run command `make linux`
### Build for Mac environments
- [x] Run command `make mac`

## How to Test
- [x] Open terminal
- [x] Add test code to `test/maintest.go`
- [x] Run command `make test`
